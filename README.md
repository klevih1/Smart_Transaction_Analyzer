# Financial Transaction Analyzer 📊

An AI-powered financial assistant that categorizes transactions, detects suspicious activity, and provides personalized spending insights. This project uses **Streamlit** for the interface and **LangChain** with **Groq (Llama 3.3)** for intelligent analysis.

## Features
- **File Upload**: Support for CSV and Excel (`.xlsx`) transaction files.
- **AI Categorization**: Automatically labels transactions (e.g., Food, Transport, Rent) using LLMs.
- **Suspicious Activity Detection**: Identifies unusually large debit transactions.
- **AI Spending Reports**: Generates a summary of spending habits and suggests two areas for improvement.
- **Data Visualization**: Displays a pie chart of spending by category.

## Prerequisites
- Python 3.8+
- A [Groq API Key](https://console.groq.com/)
